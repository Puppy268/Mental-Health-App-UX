# 🧠 Mental Health App UX Case Study

## 📌 Project Overview

This project presents the UX design process for a Mental Health mobile application focused on helping users improve their emotional well-being through meditation, mood tracking, and self-care activities.

The goal was to create an intuitive, calming, and user-friendly experience that encourages users to build healthy mental wellness habits.

---

## 🎯 Problem Statement

Many people struggle to manage stress and anxiety but find existing mental health apps overwhelming or difficult to use. The challenge was to design a simple, supportive, and engaging application that makes mental wellness accessible to everyone.

---

## 💡 Solution

Designed a modern mobile application that enables users to:

* Track daily moods
* Practice guided meditation
* Maintain a personal journal
* Set self-care goals
* Receive motivational reminders
* Monitor wellness progress

---

## 👥 Target Users

* Students
* Working professionals
* Individuals experiencing stress or anxiety
* Anyone interested in improving mental well-being

---

## ✨ Key Features

* 🔐 Secure Login & Sign Up
* 😊 Daily Mood Tracker
* 🧘 Guided Meditation Sessions
* 📖 Personal Journal
* 🎯 Self-Care Goals
* 🔔 Daily Wellness Reminders
* 📊 Progress Dashboard
